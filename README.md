# nguyenvanhuy0612.github.io

Personal blog and knowledge base for storing tutorials, guides, and notes. Built with [Jekyll](https://jekyllrb.com/) and published automatically via [GitHub Pages](https://pages.github.com/).

**Live site:** [https://nguyenvanhuy0612.github.io](https://nguyenvanhuy0612.github.io)

## Tech Stack

- **Jekyll** — static site generator (built-in GitHub Pages support, no CI needed)
- **Liquid** — templating language for dynamic page rendering
- **Kramdown** — Markdown processor
- **Rouge** — syntax highlighting for code blocks
- **Theme** — `pages-themes/minimal` (remote theme)

## Content Types

| Type | Directory | Filename format | URL |
|------|-----------|----------------|-----|
| Blog posts | `_posts/` | `YYYY-MM-DD-title.md` (required) | `/YYYY/MM/DD/title/` |
| Tutorials | `_tutorials/` | `any-name.md` | `/tutorials/title/` |
| Guides | `_guides/` | `any-name.md` | `/guides/title/` |
| Notes | `_notes/` | `any-name.md` | `/notes/title/` |

## Adding New Content

Create a Markdown file in the appropriate directory with front matter:

```yaml
---
title: "Your title here"
description: "A short description"
tags: [tag1, tag2]
---

Your content in Markdown...
```

Then push to GitHub — the site rebuilds automatically.

## Local Development

```bash
# Install dependencies (first time only)
bundle install

# Start local dev server at http://localhost:4000
bundle exec jekyll serve
```

## Design Features

- **Automatic dark mode** — uses `prefers-color-scheme` CSS media query, follows system setting
- **Responsive** — mobile-friendly layout with stacked navigation on small screens
- **No JavaScript** — pure HTML/CSS with Liquid templating
- **Code highlighting** — syntax highlighting via Rouge for code blocks
- **SEO** — meta tags generated by `jekyll-seo-tag` plugin

## Deployment

Push to the `main` branch. GitHub Pages builds and deploys automatically — no CI/CD configuration needed.
